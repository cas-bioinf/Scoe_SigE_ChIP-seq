Genome browser for S. coelicolor A3(2) with ChIP-seq for σE-HA with/without EtOH treatment

  • This is supplementary data for σE of Streptomyces coelicolor can function both as a direct activator or repressor of transcription paper
  • The raw data can be obtained from ArrayExpress (BioStudies) E-MTAB-10987
  • Samples were mapped with Bowtie2 and coverage was computed with deepTools2 bamCoverage with RPGC normalization with effective genome size computed by Kent's tools faCount.
  • Sequence and annotation for Streptomyces coelicolor A3(2) genome (AL645882.2) were obtained from NCBI assembly GCA_000203835.1
  • This presentation uses igv.js genome browser, we thank the authors.
